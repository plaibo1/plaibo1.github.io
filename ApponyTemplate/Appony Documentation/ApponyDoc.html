1. Template Structure

Before you start, launch the console in the Appony folder and type in npm i

- You can find all files in Appony folder

- In app folder you will see App, Dist and node_modules folders

- Dist folder contain all compiled and ready to use files (production)

- App folder contain all sources for your work

- CSS,Sass, js, images files you can find in App/(accordingly) folder

- node_modules folder will appear after npm i

- Also you will see gulpfile.js, package.json and package-lock.json

2. How to edit template (for users)

Better way to use sources in App folder, but if you don't know all technologies you can simply edit build files

- Change other images. Upload your image to app/img/ folder. Open index.html file and find img tag. Change the url to your image path
If you want our images contact us :)

- All texts you can change in index.html file - h1..h6 and p tags also span

How to edit sliders

- All slides marked like something__slide

- Find img tag and change the url to your image path app/img/

- All texts you can change in h1..h6 and p tags

3. How to edit template (for developers)

If you want to edit some html, css, js files, best way to use sources in App/(accordingly) folder. You need node.js, gulp and bower installed

- Run npm i to install all gulp dependencies

- Run gulp default to run local server and start watching changes

- Run gulp watch to watch changes

- Run gulp sass to convert sass to css

- Run gulp jsFiles to to build only js files

- Run gulp cssFiles to to build only css files

- Run gulp doFiles to to build js & css files

- Run gulp img to build only images and run gulp clearCache to clear img

- Run gulp doBuild to build project "dist"

- Run gulp clean to delete build folder

How to work with html, css, js

- To change html open app/index.html/ file - code is well documented

- To change css open app/sass/main.sass file - code is well documented

- To change js open app/js/common.js file - code is well documented

Libs folder

- All librarians are in app/libs folder

- All paths to libs are in Appony/gulpfile.js 41-45 strings, You can add yours libs (code is documented)

- To add your lib use npm install -g bower in cmd, after you will see bower_components folder

- Install your lib, bower docs here

- Drag all files bower_components folder to app/libs and tune paths like Appony/gulpfile.js 41-45 strings

How to edit swiper slider
